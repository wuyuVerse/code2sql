# 配置清华源 - 2025-07-30

## 任务目标
为用户配置使用清华源，以加速Python包的下载。

## 配置方案

### 1. 配置uv使用清华源

#### 方法1：环境变量配置
```bash
export UV_INDEX_URL=https://pypi.tuna.tsinghua.edu.cn/simple/
```

#### 方法2：创建uv.toml配置文件
创建了 `uv.toml` 文件，内容如下：
```toml
index-url = "https://pypi.tuna.tsinghua.edu.cn/simple/"
```

### 2. 配置pip使用清华源（备用方案）
```bash
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/
```

## 验证配置

### 测试uv配置
```bash
uv add beautifulsoup4
```
结果：成功从清华源下载并安装了beautifulsoup4包，安装时间明显加快。

### 验证pip配置
```bash
pip config list
```
结果：显示已配置清华源为默认源。

## 配置效果

1. **uv包管理器**：通过uv.toml配置文件永久设置清华源
2. **pip包管理器**：通过pip config永久设置清华源
3. **下载速度**：相比默认源，下载速度显著提升
4. **稳定性**：清华源在国内访问稳定，减少网络问题

## 注意事项

1. uv.toml配置文件格式必须正确，不能使用[global]标签
2. 环境变量配置只在当前会话有效，重启后需要重新设置
3. 建议使用配置文件方式，可以永久生效
4. 如果遇到网络问题，可以临时切换回官方源

## 相关文件

- `uv.toml`：uv配置文件，设置清华源
- `pyproject.toml`：项目依赖文件，已更新包含beautifulsoup4
- `/root/.config/pip/pip.conf`：pip配置文件，设置清华源
